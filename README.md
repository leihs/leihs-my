### Formatting Code

#### Clojure

Use `./bin/cljfmt check` and `./bin/cljfmt fix`.

From vim you can use `:! ./bin/cljfmt fix %` to format the current file.

#### Ruby

Use `./bin/rblint` and `./bin/rblint --fix`.
